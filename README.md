# Opti-Sphere Project

---

## Installation

1. Clone this repository 
```
git clone https://github.com/maximeduby/Opti-Sphere.git
```
2. Change directory
```
cd Opti-Sphere/
```
3. Install required libraries
```
pip install -r requirements.txt
```
4. Run the program
```
python opti-sphere/app.py
```
